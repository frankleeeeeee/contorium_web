MCP compatible

Works with modern AI agents

Contorium exposes persistent workspace memory through MCP-compatible tools—so Claude Code, Cursor, OpenAI agents, and MCP clients can retrieve context on demand.

  • Claude Code
  • Cursor
  • OpenAI Agents
  • MCP clients

Quick start (after npm run build:mcp)
# Build first (repo root)
npm run build:mcp

# Claude Code — plugin
claude --plugin-dir .

# Claude Code — MCP only (project scope)
claude mcp add --scope project contorium \
  -- node ./packages/mcp/dist/server.js

# Cursor — root mcp.json or MCP settings

Tools: get_workspace_context, store_memory, search_memory, get_memory. Requires the Contorium extension writing .contora/state.json.
